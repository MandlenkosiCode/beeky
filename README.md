# Beeky — Electron Desktop Pomodoro App

Beeky is a calm, minimalist Pomodoro timer desktop application built with  
**Electron + React + TypeScript + Vite + Tailwind**.

The goal of this app is to provide a light, distraction-free, zen-like experience  
for focused work and healthy breaks.

## Features (planned)

- Pomodoro / Short Break / Long Break cycles
- Zen-inspired minimal UI
- Customizable durations
- Sounds & desktop notifications
- Stats and session history
- System tray controls
- Cross-platform builds (Windows / macOS / Linux)

## Tech Stack

- **Electron** (desktop runtime)
- **React + TypeScript**
- **Vite** (fast dev server)
- **Tailwind CSS**
- **electron-builder** (packaging)
